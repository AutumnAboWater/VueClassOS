# 部署本地/快速开始/安装起步
* 1.下载或git clone项目
* 2.cd to-be-number-one-master
* 3.npm i 安装依赖
* 4.cd public  npm install -g json-server 安装json-server插件
* 5.cd public json-server Data.json 运行虚拟数据
* 6.npm run dev 运行项目
* 7. http://localhost:5173/ 打开测试
  8. 测试默认用户名：admin 密码：admin
